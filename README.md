# KidsTV Pad 项目文档

## 1. 项目概述

**KidsTV Pad** 是一款专为 iPad 设计的儿童电视应用。其核心理念是提供一个由家长掌控的纯净观看环境，家长能够添加频道，并控制“频道”里边的内容，儿童也能在其间自由选择。模拟传统电视体验，让孩子通过简单的"换台"方式观看视频。

*   **目标用户**：儿童（观看者）与家长（管理者）。
*   **使用场景**：在 iPad 上模拟传统电视体验。
*   **交互设计**：极简的前台（儿童）与功能完整的后台（家长）。
*   **核心痛点**：解决三个问题：视频应用界面混乱、低质量视频横行、广告不分年龄；家长想让儿童看的东西难以选择；儿童要么没有任何选择的自由度，全靠家长控制，要么在低质量乱七八糟的视频中不受控。

## 2. 核心功能需求

### 2.1 前台界面（儿童使用）

**播放逻辑**
*   **断点续播**：切换频道回来时，自动从上次播放的进度继续播放。
*   **自动连播**：当前视频结束后，自动播放列表中的下一个视频。
*   **启动恢复**：应用启动时，自动进入上次观看的频道。
*   **更多功能**：可以查看频道里的视频列表。

### 2.2 后台管理界面（家长使用）

**访问控制**
*   **进入方式**：点击管理按钮，输入密码进入后台。

**视频库管理 (Video Library)**
*   **独立模块**：视频导入功能与频道创建分离。

采用类似 Infuse 的**媒体库 (Media Library)** 架构：

*   **媒体源 (Sources)**：用户添加的根目录位置。
*   **支持范围**：支持导入本地目录、WebDAV、SMB、阿里云盘、百度网盘、115网盘、Emby、Jellyfin
*   **扫描与索引 (Indexer)**：应用后台遍历这些源，发现所有视频文件，自动提取元数据（时长、分辨率），生成缩略图，并存入本地数据库
*   **视频库 (Library)**：所有被索引视频的集合池。
*   **一次配置，自动更新**：只需配置一次 NAS 目录，往目录里扔新视频，App 自动扫描入库，无需手动逐个添加。
*   **无需下载**：支持在线播放，无需下载到本地。
---

**频道管理**
*   **创建/编辑**：设置频道名称、图标、默认音量。
*   **内容选择**：从“视频库”中勾选该频道包含的视频。支持语义化搜索、逐个添加、以及多选。
*   **排序**：调整频道顺序。

**设置与管控**
*   **时长控制**：每日限额、时间段限制、超时锁定。
*   **全局音量限制**。
