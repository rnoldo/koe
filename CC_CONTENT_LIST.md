### 技术实现方式

这些内容可以通过以下方式集成到 App 中：

1. **内置 Index 文件**：App 内置一个 JSON 索引，包含所有 CC 内容的 Internet Archive 直链（archive.org 支持直接流播放）
2. **不打包视频文件**：视频从 archive.org 在线流播放，不增加 App 体积
3. **用户可删除**：预设频道可由家长在后台删除或编辑
4. **定期更新**：通过远程 JSON 更新索引，新增进入公共领域的内容（每年 1 月 1 日有新作品到期）

```
示例 JSON 索引结构：
{
  "version": 1,
  "channels": [
    {
      "name": "大力水手",
      "icon": "figure.sailing",
      "videos": [
        {
          "title": "Popeye Meets Sindbad",
          "year": 1936,
          "duration": 960,
          "stream_url": "https://archive.org/download/popeyethesailor-publicdomain/...",
          "thumbnail_url": "https://archive.org/services/img/..."
        }
      ]
    }
  ]
}
```

## 五、版权注意事项

1. **公共领域 ≠ 全球通用**：以上版权状态基于美国法律。部分作品在其他国家可能仍受版权保护（如迪士尼作品在欧盟的保护期更长）
2. **商标仍然有效**：角色名称和形象可能仍受商标保护（如 Mickey Mouse、Popeye）。App 中应使用"经典动画"等通用描述，避免将角色名用于商业推广
3. **逐年扩充**：每年 1 月 1 日，95 年前发布的作品进入公共领域。2027 年将有 1931 年的作品到期，届时会有更多经典内容可用
4. **Blender CC-BY**：使用 Blender 动画需注明出处（如在 App 的关于页面中注明"动画由 Blender Studio 制作，CC-BY 许可"）
